# Task Manager Web App

A simple task manager built with HTML, CSS and JavaScript. Tasks are stored in the browser's `localStorage`.

## Tech stack

- HTML
- CSS
- Vanilla JavaScript

## How to run

Just open `index.html` in your browser (double-click or open with a simple static server).

## Features

- Add tasks
- Mark tasks as completed (click on the text)
- Delete tasks
- Persistent storage using `localStorage`
