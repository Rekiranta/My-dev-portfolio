# DevOps CI Demo (Flask + Pytest + GitHub Actions)

A minimal Python API with automated tests and a CI pipeline.

## Tech stack

- Python
- Flask
- Pytest
- GitHub Actions (CI)

## Run locally (Windows / PowerShell)

Create a virtual environment and install dependencies:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
```
