# Expense Tracker Dashboard (Chart.js)

A visually polished expense tracker that stores data in `localStorage` and shows a bar chart of expenses per category.

## Tech stack

- HTML
- Modern CSS (dashboard layout)
- Vanilla JavaScript
- Chart.js (via CDN)

## How to run

You can simply open `index.html` in your browser.

For local development with a static server (optional):

```bash
# For example with Python
python -m http.server 8000
```
