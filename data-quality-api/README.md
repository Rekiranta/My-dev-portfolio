# Data Quality API & Analyzer

A simple but realistic data engineering project demonstrating:

- Data ingestion (API)
- Data validation and quality checks
- SQLite storage
- Automatic QC reports (missing values, negatives, duplicates)
- Web dashboard

## Tech stack

- Python
- Flask
- SQLite
- HTML/JS

## How to run

```bash
pip install -r requirements.txt
python app.py
```
